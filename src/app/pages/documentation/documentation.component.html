API TTS Darija

Endpoint de base

POST https://api.sawtai.com/v1/tts/synthesize

Paramètres

text
string
Texte en Darija à synthétiser (requis)
voice
string
ID de la voix (homme/femme)
speed
number
Vitesse de lecture (0.5 à 2.0)
format
string
Format audio (mp3, wav, ogg)

Exemple de requête

curl -X POST https://api.sawtai.com/v1/tts/synthesize \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{{'{'}}"text": "مرحبا بيك فالموقع ديالنا",
      "voice": "female_01",
      "speed": 1.0,
      "format": "mp3"{{'}'}}' 
  --output audio.mp3

API Chatbot IA

Endpoint de base

POST https://api.sawtai.com/v1/chatbot/message

Paramètres

message
string
Message de l'utilisateur (requis)
session_id
string
ID de session pour le contexte
language
string
Langue (darija, français, arabe)
voice_response
boolean
Retourner une réponse audio

Exemple de requête

curl -X POST https://api.sawtai.com/v1/chatbot/message \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{{'{'}}"message": "شحال الساعة دابا؟",
      "session_id": "user_123",
      "language": "darija",
      "voice_response": true{{'}}'}'

Authentification

Utilisation de la clé API

Toutes les requêtes doivent inclure votre clé API dans l'en-tête Authorization :

Authorization: Bearer YOUR_API_KEY

Sécurité

  • Ne partagez jamais votre clé API secrète
  • Stockez vos clés dans des variables d'environnement
  • Utilisez HTTPS pour toutes les requêtes

Exemples de Code

JavaScript / Node.js

const Sawtai = require('@sawtai/sdk');

const client = new Sawtai({{'{'}}"apiKey": process.env.SAWTAI_API_KEY{{'}'}});

// TTS Example
const audio = await client.tts.synthesize({{'{'}}"text": "مرحبا بيك",
  "voice": "female_01"{{'}'}});

// Chatbot Example
const response = await client.chatbot.sendMessage({{'{'}}"message": "كيف نقدر نعاونك؟",
  "sessionId": "user_123"{{'}'}});

Python

from sawtai import SawtaiClient

client = SawtaiClient(api_key=os.environ.get('SAWTAI_API_KEY'))

# TTS Example
audio = client.tts.synthesize(
    text="مرحبا بيك",
    voice="female_01"
)

# Chatbot Example
response = client.chatbot.send_message(
    message="كيف نقدر نعاونك؟",
    session_id="user_123"
)

Guide d'Intégration

Intégration Web

Ajoutez facilement nos services à votre site web :

<script src="https://cdn.sawtai.com/sdk.js"></script>
<script>
  Sawtai.init({{'{'}}"apiKey": "YOUR_PUBLIC_KEY"{{'}'}});
  
  // Utiliser le TTS
  Sawtai.speak("مرحبا بيك");
</script>

Intégration Mobile

SDKs disponibles pour iOS et Android

// iOS (Swift)
import SawtaiSDK

let client = SawtaiClient(apiKey: "YOUR_API_KEY")
client.tts.speak(text: "مرحبا بيك")

// Android (Kotlin)
import com.sawtai.sdk.SawtaiClient

val client = SawtaiClient("YOUR_API_KEY")
client.tts.speak("مرحبا بيك")

Questions Fréquentes

Quelles langues sont supportées ?

Nous supportons principalement le Darija (arabe marocain), mais aussi le français et l'arabe standard. Plus de 50 langues sont disponibles pour le chatbot.

Quelles sont les limites d'utilisation ?

Les limites dépendent de votre plan :

  • Starter : 10 000 requêtes/mois
  • Professional : 100 000 requêtes/mois
  • Enterprise : Illimité

Comment gérer les erreurs ?

Notre API retourne des codes HTTP standards :

200 - Succès
400 - Requête invalide
401 - Clé API invalide
429 - Limite de requêtes atteinte
500 - Erreur serveur
